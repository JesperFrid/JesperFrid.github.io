1. Rastergrafik använder sig av pixlar för att skapa en bild. Används ofta inom fotografier för detaljrika bilder. Vektorgrafik använder sig av matematiska banor för att bilda en bild. Det gör att bilder kan skalas upp utan att förlora kvalite. Detta används vid logotyper och ritningar
  2. Några exempel är jpg, png och gif. Jpg kan innehålla metadata och förlorar kvalite vid komprimering. Png kan ha bra komprimering, väldigt många färger och kan även innehålla metadata. Gif är ett fil format som använder sig av många olika bilder för att skapa en animation.
